Aquest repositori es per generar els punts de mostreig pel projecte de seguiment dels habitats de Catalunya. 
Està pensat per ser utilitzat a partir d'una capa de polígons amb els atributs de cada polígon els quals poden ser codi per HIC, CORINE, Llista Vermella etc. 
## Com funciona:
1. Els scripts tenen un número davant, el qual es l'ordre en que s'han d'executar
2. Els resultats es desen en carpetes amb el mateix nom que l'script que els genera.
