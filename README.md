# Seguiment dels habitats a Catalunya
Aquest repositori es per generar els punts de mostreig pel projecte de seguiment dels habitats de Catalunya. 
Està pensat per ser utilitzat a partir d'una capa de polígons amb els atributs de cada polígon els quals poden ser codi per HIC, CORINE, Llista Vermella etc. L'objectiu es generar 30 punts per categoria els quals estiguin com a mínim 200m i representin la distribució espacial de l'habitat de la millor manera possible. 
## Com funciona:
1. Els scripts tenen un número davant, el qual es l'ordre en que s'han d'executar
2. Els resultats es desen en carpetes amb el mateix nom que l'script que els genera.
